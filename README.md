# Multi-sensor-fusion-SLAM-with-lidar-camera-and-IMU
This project is based on the Cartographer algorithm and uses LiDAR, depth cameras, and IMUs for localization and mapping. The IMU and wheel odometers are fused using extended Kalman filtering for localization, while the depth camera is fused with the LiDAR through depth-to-light conversion. The navigation module is based on navigation2.
